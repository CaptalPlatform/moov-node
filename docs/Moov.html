The Moov API client.

Constructor

new Moov(accountID, publicKey, secretKey, gotInstanceopt)

Initializes a new instance of Moov.

Moov uses the Got HTTP client library. If you need to access or customize the request-response pipeline, provide an instance of Got in the gotInstance parameter.

Customized Got instance. See docs.
